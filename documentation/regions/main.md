# Region

Regions are split areas the world is divided. When a person enters a specific regions some
actions occurs. For example, the [Veins](../gathering/main.md) are spawned.

```
{
   "_id": ObjectId;
   "x1": Int32;
   "z1": Int32;
   "x2": Int32;
   "z2": Int32;
   veinsLocations: [VeinLocation]
}
```
